Coleção de plugins para RPG Maker MV desenvolvidos com foco em qualidade, performance e facilidade de uso.

## Como Instalar

1. Baixe o arquivo .js do plugin desejado
2. Coloque o arquivo na pasta js/plugins do seu projeto
3. Abra o Plugin Manager no RPG Maker MV
4. Ative o plugin e configure os parâmetros conforme necessário
5. Salve o projeto e teste o jogo

Recomenda-se testar os plugins em um projeto de teste antes de aplicar no projeto principal.

## Compatibilidade

- RPG Maker MV (versão 1.6.x recomendada)
- Alguns plugins podem funcionar no MZ com ajustes
- Compatibilidade com outros plugins está informada na documentação de cada um
